Как всё должно функционировать (для создания работы):
При запуске программы предлагается создать новую работу.
Потом появляется новое окно с плюсом по середине. После нажатия на него вылетает новое окно, где надо выбрать тип текущего задания.
После выбора типа внизу появляется кнопка-круг (или в ином виде) с цифрой, обозначающая, что уже создано задание, которое можно изм.
После каждого добавления нового задания появляются новые цифры (ограничить кол-во заданий)

Добавить кнопку сохранения работы. По её нажатию в файл (предположительно на данный момент формата txt) будет загружена информация по проету.
Она будет представлять это:
В первой строке количество заданий, их типы (предположительно будет 1) со своим ответом 2) с предложенными вариантами)
далее в N блоках (каждому типу заданий свой размер блока, измеряется в строках) будут храниться сами задания, потом в след строке ответ,
в остальных строках варианты ответов (если соответствует задание)
